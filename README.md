VeraCrypt
=========

Install VeraCrypt in Debian 10 "Buster"

Requirements
------------

No Requirements.

Role Variables
--------------

Default ones should be OK.

Dependencies
------------

No dependencies.

Example Playbook
----------------

    - hosts: localhost
      connection: local
      roles:
        - { role: ansible-role-VeraCrypt }

Run the playbook with: "ansible-playbook VeraCrypt.yml"
